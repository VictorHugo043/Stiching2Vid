# 05_evaluation

## 任务目标
- 建立拼接质量与效率评估体系：对齐误差、接缝可见性、颜色一致性、时间稳定性、运行效率。

## 验收标准(DoD)
- 至少实现 3 类指标并在样例视频上输出数值结果。
- 产出基础对比表或可视化图。

## 下一步
- 引入更强匹配方法进行对比（见 `06_method2_strong_matching`）。

## 技术方案
- 对齐误差：重投影误差或边界偏差。
- 接缝可见性：梯度变化或亮度差统计。
- 时间稳定性：重叠区帧间差异与单应参数抖动。

## 关键决策
- 指标优先选择可实现且可解释的方案。
